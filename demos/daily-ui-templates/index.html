日常高频 UI 模板库 Demo

先按你日常真实工作流规划模板类型，再生成一组差异化 Demo。当前只做候选页面，不直接拆进组件库；你确认喜欢的方向后，再按组件粒度拆解、入库、生成缩略图。

设计约束

  • 不用 emoji 图标，优先使用版式、线条、编号、几何图形表达信息层级。
  • 避免同质化：每个 Demo 使用不同视觉语言和页面结构。
  • 优先覆盖日常高频：规划、评测、情报、资产、项目、计分。
  • 可拆解为组件：每个页面都内含可复用模块。
